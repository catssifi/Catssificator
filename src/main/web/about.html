Missionary
Think about it would be nice to have an automatic intelligent system that can classify all your search input into different categories without having a nightmare to develop one on your own (risks/resources/budgets..etc ?). Your first thought of solution might be Google - well, however, this is not a nice free lunch. Once you send them lot of 'robot' alike queries, they are just smart enough to figure out you are not a normal user and will block your IP in a second. Therefore, it is always our mission to provide an free open-source A.I. system that can do this job right for you - you teach the system to recongize a query to a category, then next time you ask the same query again the system will know what category (or similar categories)should belong to the query!
Why Catssificator
If you are from any marketing firm or advertising or even any organization firm who wants to analyze your audiences' search activities and categorize these into different domains, then Catssificator is the right solution for you. It provies you with a web-based platform and a powerful REST-ful API to interact. As of now, your interactions (such as submitting a new query to a category, uploading a file to a category...etc) will be recorded and available to anyone else. It is like Wikipedia - a public free repo for knowledge!
Technically, what makes it unique to other samilar famous open-source product(s)
Elastic Search (E.S.): First, E.S. does not have a capability to learn from the past. Second, and most importantly, given a very small search query (like few words), E.S. is good at finding which content(s) this small search query belongs to. However, we are exactly doing the opposite: we can take a very very long query (like hundreds or thousands words and deduce it to a specify category(ies) which usually has only few words. Therefore, to conclude, E.S. specalizies at solving the search query problems while we try to solve the data categorization or classification problems.
